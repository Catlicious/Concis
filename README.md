## Getting Started

Install dependencies,

```bash
$ npm i react-view-ui & yarn add react-view-ui
```

Import the UI,

```bash
$ import { Button } from 'react-view-ui/web-react';
$ import 'react-view-ui/web-react/style/index.css'
```

Use,

```bash
$ <Button type="primary">基础按钮</Button>
```
